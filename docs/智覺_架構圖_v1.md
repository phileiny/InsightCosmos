# 智覺個人版 1.0 架構圖（Markdown 版）

## 系統總覽圖
```
[Scout] → [Analyst] → [SQLite/Embedding Memory] → [Curator Daily/Weekly] → [Email Delivery]
```

## Agents
- Scout: RSS + Google Search 爬取
- Analyst: LLM 摘要與分析
- Curator: Daily / Weekly 報告
- Memory: SQLite + Embedding

## Pipelines
### Daily
RSS → 分析 → 儲存 → Daily Digest → Email

### Weekly
DB → 趨勢分析 → Weekly Report → Email

（此為簡化版架構圖）
